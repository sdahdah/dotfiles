# dotfiles

Stow-based repository containing my Linux configuration files.

## Instructions

To deploy a set of dotfiles (*e.g.*, `.bashrc` and `.bash_profile`) run
```
$ cd ~/.dotfiles
$ stow bash
```

If any of the configuration files already exist, you will have to delete them first.
